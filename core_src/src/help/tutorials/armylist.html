Introduction

First you have to get eve a bit familiar with the data structure the software uses to describe an army list. Don't worry. No need for technical details.


Components in a Army List

This section gives a short explanation about each of the different components in an army list.

Army List

This is the main component. An armylist contains a number of units, wargear and weapons.

Unit

A unit is the key component An army created from this list is composed from units. Unit is the only component that can be added to a army.



Model

A unit is composed from models.



Unit/Model Update

Updates modify units/models equipment.



Wargear

Wargear is grouped by wargear type. Any model can be given access to wargear groups.



Weapon

Wargear or update can be a weapon. Weapons doesn't have point cost. The point cost should be included into the update or wargear. Weapons are optional. Army list doesn't have to include any weapons.
